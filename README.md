 # Projeto Receitas Culinárias

Este projeto foi desenvolvido para praticar HTML, CSS e Git, criando páginas de receitas com estrutura organizada e estilização visual.

## Como usar

Abra o arquivo `index.html` em seu navegador para visualizar a lista de receitas disponíveis.

## Estilização

O projeto utiliza um arquivo CSS localizado em `css/style.css` para aplicar estilos visuais às páginas.
